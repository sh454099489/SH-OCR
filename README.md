# SH-OCR

欢迎来到SH-OCR!

SH-OCR是一个文档处理和文本识别工具，希望您能喜欢我的工作！

**2020/08/12 第一次更新！**

**印章去除工具**

添加了一个印章去除工具，它可以在去除印章的同时，保留被章覆盖的字。

requrements:

python 3
opencv 3.0+
numpy

参数说明：

参数：img 接受一个图片路径或者OpenCV读取的图片对象

参数：thr 分割字体与章的灰度阈值

参数：point 设置一个像素的坐标，根据这个坐标的像素值，生成结果图片
